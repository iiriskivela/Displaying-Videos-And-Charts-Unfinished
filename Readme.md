# Unfinished Project for Students

This project is an exercise for students to complete. Your task is to finish the implementation of two features by following the `TODO` comments in the code.

## 1. Feature: Video Display

This feature involves displaying a list of videos and allowing users to upload their own.

### Files to complete:

*   `app/src/main/java/com/example/learningdashboard/ViewModels/SharedViewModel.kt`
*   `app/src/main/java/com/example/learningdashboard/screens/VideoScreen.kt`
*   `app/src/main/java/com/example/learningdashboard/screens/UploadVideoScreen.kt`

## 2. Feature: Charts

This feature involves displaying charts that visualize user data.

### Files to complete:

*   `app/src/main/java/com/example/learningdashboard/screens/ProgressScreen.kt`
*   `app/src/main/java/com/example/learningdashboard/DailyUsageBarChart.kt`
*   `app/src/main/java/com/example/learningdashboard/DailyUsagePieChart.kt`

**Instructions:**

*   Each file contains `TODO` comments that will guide you through the implementation.
*   You should not need to modify any other files to complete the exercises.
